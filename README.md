# 🚀 Data Cleaning & Analytics Pipeline
**Developer:** Vansh Raj  
**Environment:** Pydroid 3 (Mobile Development)

### 📌 Project Overview
This project is a professional-grade Python automation tool designed to transform "dirty" raw data into structured, actionable sales reports. 

### 🛠 Key Features
* **Data Ingestion:** Seamlessly reads CSV and Excel datasets using `Pandas`.
* **Automated Cleaning:** * Identifies and removes duplicate entries.
    * Trims whitespace and standardizes text formatting.
* **Advanced Analytics:** Calculates Total Revenue and Average Order Value (AOV).

### 📊 Sample Output
```text
==============================
      SALES REPORT      
==============================
Total Revenue: 3983.33
Average Order Value: 569.05
==============================
